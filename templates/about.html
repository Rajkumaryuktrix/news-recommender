About This Tool

This is a dummy website for testing news article recommender systems and running A/B tests.

There are 5,000 users. Each user has a different reading history. You can log in as a specific user with the same credential format as user 0: User Name = user0, Password = userpassword

Data for this website comes from Microsoft's recommenders library for Python. The news articles come from Microsoft's MIND dataset.
